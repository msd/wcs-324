# wcs-324

university coursework on computer graphics, tetris-clone using opengl

## Prerequisites

* CMake
* GNU c++ compiler
* [GLEW](http://glew.sourceforge.net/)
* [GLM](http://glm.g-truc.net/0.9.9/index.html)
* [freeglut](http://freeglut.sourceforge.net/)

## Compile

`bash unixbuild.sh`

## Run

`build/tetris`
